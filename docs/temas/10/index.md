# **UT10: Interfaces gráficas**

!!! tip "Información de la unidad"

    === "Contenidos"

        Desarrollo de interfaces gráficos de usuario (GUI):

        - Herramientas del entorno de desarrollo para el diseño de interfaces.
        - Componentes visuales básicos (botones, etiquetas, campos de texto, etc.).
        - Concepto de eventos y controladores de eventos.
        - Programación de eventos para interacción con el usuario.
        - Uso de interfaces gráficos para entrada y salida de información.

    === "Propuesta didáctica"

        En esta unidad vamos a seguir trabajando el **RA5: Realiza operaciones de entrada y salida de información, utilizando procedimientos específicos del lenguaje y librerías de clases.**

        Criterios de evaluación clave que abordaremos:

        - **CE5f**: Se han utilizado las herramientas del entorno de desarrollo para crear interfaces gráficos de usuario simples.
        - **CE5g**: Se han programado controladores de eventos.
        - **CE5h**: Se han escrito programas que utilicen interfaces gráficos para la entrada y salida de información.

    === "Planificación de Aula"

        Esta unidad se imparte en la **tercera evaluación**, con una duración estimada de **22 sesiones lectivas**, aproximadamente entre la **4ª semana de marzo y la 3ª semana de abril de 2026**
