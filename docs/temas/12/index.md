# **UT12: Acceso a BBDD relacionales**

!!! tip "Información de la unidad"

    === "Contenidos"

        Acceso a Bases de Datos Relacionales:
        - Características y métodos de acceso a sistemas gestores de bases de datos relacionales.
        - Conexión con bases de datos.
        - Programación para almacenar información en bases de datos (INSERT).
        - Programación para recuperar y mostrar información (SELECT).
        - Programación para efectuar borrados (DELETE) y modificaciones (UPDATE) sobre la información.
        - Creación de aplicaciones para mostrar y gestionar información presente en bases de datos relacionales.

    === "Propuesta didáctica"

        En esta unidad vamos a trabajar el **RA9: Gestiona información almacenada en bases de datos manteniendo la integridad y consistencia de los datos.** (Este RA es "Dual", indicando su importancia y posible aplicación en diferentes contextos).

        Criterios de evaluación clave que abordaremos:

        - **CE9a**: Se han identificado las características y métodos de acceso a sistemas gestores de bases de datos.
        - **CE9b**: Se han programado conexiones con bases de datos.
        - **CE9c**: Se ha escrito un código para almacenar información en bases de datos.
        - **CE9d**: Se han creado programas para recuperar y mostrar información almacenada en bases de datos.
        - **CE9e**: Se han efectuado borrados y modificaciones sobre la información almacenada.
        - **CE9f**: Se han creado aplicaciones que muestren la información almacenada en bases de datos.
        - **CE9g**: Se han creado aplicaciones para gestionar la información presente en bases de datos.

    === "Programación de Aula"

        Esta unidad se cubre durante el período de **prácticas en la empresa**, con una duración de **28 sesiones lectivas**, aproximadamente desde el **11 de mayo hasta el 5 de junio de 2026**. Durante este periodo, los alumnos aplicarán y consolidarán estos conocimientos en un entorno real.

        | Sesión | Contenidos (Aplicación en la empresa)                                                                   | Criterios trabajados |
        | ------ | ------------------------------------------------------------------------------------------------------- | -------------------- |
        | 1-4    | Familiarización con los sistemas de BBDD de la empresa. Conexión con BBDD.                              | CE9a, CE9b           |
        | 5-8    | Escritura de código para almacenar y recuperar información (`INSERT`, `SELECT` básicos).                | CE9c, CE9d           |
        | 9-12   | Modificación y borrado de información (`UPDATE`, `DELETE`).                                             | CE9e                 |
        | 13-16  | Consultas avanzadas: `WHERE`, `ORDER BY`, `GROUP BY`, funciones de agregación.                          | CE9d                 |
        | 17-20  | Joins, subconsultas y vistas.                                                                           | CE9d                 |
        | 21-24  | Gestión de transacciones y control de integridad.                                                       | CE9c, CE9e           |
        | 25-28  | Creación y mantenimiento de aplicaciones de gestión de datos, informes o APIs que interactúen con BBDD. | CE9f, CE9g           |
